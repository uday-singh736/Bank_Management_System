# Bank Management System

## 📌 Description
This is a simple Bank Management System built using Python and MySQL.  
It allows users to create accounts, store details, and manage balances.

## 🚀 Features
- Create new account
- Store user details
- Manage account balance
- Simple and interactive interface

## 🛠️ Tech Stack
- Python
- MySQL

## ▶️ How to Run
1. Install Python
2. Install MySQL
3. Run the program:
   python main.py
4. Enter your MySQL credentials when asked

## ⚠️ Important
- Make sure MySQL is running
- User should have permission to create database

## 📂 Files Included
- main.py (Main program)
- database.sql (Database structure)
